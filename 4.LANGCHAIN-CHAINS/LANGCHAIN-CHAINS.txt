## 📝 Sequential Chain

* Executes components **one after another**.
* Output of one step becomes the input of the next.
* Used when later tasks depend on earlier results.
* Operator: `|`

```
Input
  │
Prompt
  │
Model
  │
Parser
  │
Output
```

**Example:** Translate → Summarize → Explain

---

## 📝 Parallel Chain

* Executes **multiple chains simultaneously**.
* All branches receive the **same input**.
* Produces multiple outputs at once.
* Uses `RunnableParallel`.

```
          Input
            │
    ┌───────┼────────┐
    │       │        │
 Summary  Keywords Sentiment
    │       │        │
    └───────┼────────┘
         Combined Output
```

**Example:** From one article:

* Summarize
* Extract keywords
* Detect sentiment

---

## 📝 Conditional Chain

* Chooses **which chain to execute** based on a condition.
* Only **one branch** runs.
* Uses routing logic (`RunnableBranch` or custom logic).

```
            Input
              │
         Check Condition
         /             \
      True             False
       │                 │
   Chain A           Chain B
       │                 │
       └──────► Output ◄──┘
```

**Example:**

* Positive review → Thank the customer
* Negative review → Apologize and escalate

---

## 🚀 Quick Comparison

| Chain Type      | Execution         | Use Case                            |
| --------------- | ----------------- | ----------------------------------- |
| **Sequential**  | One after another | Multi-step workflows                |
| **Parallel**    | Simultaneously    | Independent tasks on the same input |
| **Conditional** | One selected path | Decision-making / Routing           |

---

### 🧠 Easy way to remember

* **Sequential** = **Do this, then that.**
* **Parallel** = **Do everything at the same time.**
* **Conditional** = **If this, do A; otherwise, do B.**

This is the core of LCEL. Once you move into LangGraph, you'll see these same three ideas become graph nodes and edges, with LangGraph adding loops, state management, and more advanced control flow.
